# Contributing to Echo's Bigger Workbenches

Thank you for wanting to help improve the mod! 

Please note that this project is strictly **All Rights Reserved**. By contributing 
to this project via Pull Requests, you agree that your code will be integrated 
into the main project under the core proprietary terms, granting the author 
the right to use, distribute, and modify your contribution indefinitely. 

### Guidelines & Technical Rules

1. **Environmental Fit:** Modified workbenches must fit cleanly within their environments without clipping through static terrain. (Clipping through dynamic, moving objects like hangar doors is perfectly acceptable).
2. **Preserve Structure Workbenches:** Structure workbenches must remain entirely untouched to maintain perfect compatibility with the Steam Workshop.
3. **Maintain Early Game Balance:** Starter and beginner bases must remain untouched to retain a reasonable sense of progression and balance.
4. **Testing:** Always fully test your modified workbench files in Stormworks before submitting a Pull Request to ensure they load and function correctly.
5. **Documentation & Conduct:** Ensure any changes are clearly documented in your PR description. Please remain respectful in commit histories, pull requests, and the issue tracker.
